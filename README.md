# Word Dictionary

A Codex skill that explains the meaning of a selected, highlighted, quoted, or otherwise indicated word or short phrase.

## Use This Skill

Use `define-selected-word` when you want Codex to explain a word in plain language, especially when the word appears in a sentence or document.

Example prompts:

- `What does "resilient" mean?`
- `Define the selected word.`
- `Explain "bear market" in this sentence.`

The skill gives a concise meaning, part of speech when useful, an example sentence, and context-specific notes when the surrounding text changes the meaning.
